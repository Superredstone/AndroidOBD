## Android OBD Library


### What is this repository for? ###

This project offers a developer friendly interface to communicate with ELM 327 OBD devices via BLUETOOTH.

### Usage ###

To use:
```Gradle
implementation 'com.pnuema.android:obd:1.0.2'
```
